\subsection{Handy Commands}\label{app:Temp}
One can use \textbackslash gls to utilize 
the glossary package like this: 
\gls{gls}.\\
One can use \textbackslash hltodo \hltodo{to mark text}{and add comments to 
it}.\\
One can use \textbackslash autoref to reference sections and subsections that 
have a label. \autoref{sec:prerequisite}. \\
One can use \textasciitilde \textbackslash cite to cite sources~\cite{MyAss}. \\
One can also use \textbackslash footnote to\footnote{make footnotes.}
