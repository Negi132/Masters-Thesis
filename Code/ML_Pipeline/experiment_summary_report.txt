METHODOLOGICAL NOTE ON METRICS AND DELTA TARGETS:
When predicting 'Delta' (the hour-to-hour change in price), percentage-based
metrics like WMAPE mathematically explode. Because the actual target values
are clustered near zero, dividing any error by a near-zero denominator
creates an artificially massive percentage.

To ensure a true 1:1 scientific comparison between Absolute Price targets
and Delta targets, this summary ranks all experiments using Mean Absolute
Error (MAE). MAE represents the average error in raw Euros, stripping away
percentage distortions and revealing the true predictive accuracy.
